## üìä Damage Assessment Methodology

### Conceptual Framework

The economic damage assessment is based on the **replacement cost method**, which involves calculating the costs to restore the ecosystem to its original state.

### Damage Components

| Component | Description | Assessment Method |
|-----------|-------------|-------------------|
| **Vegetation Damage** | Loss of biomass, forest fires | Restoration cost per hectare |
| **Soil Degradation** | Mechanical destruction, pollution | Soil reclamation cost |
| **Water Pollution** | Contamintation with toxic fuel components | Water purification cost |
| **Mechanical Damage** | Crater formation, infrastructure destruction | Restoration cost |
| **Indirect Losses** | Loss of ecosystem services, reduced productivity | Benefit transfer method |

## üßÆ Mathematical Cost Calculation Models

### Basic Equations

#### 1. Vegetation Restoration Cost (C·µ•)

$$
C_v = \sum_{i=1}^{n} A_i \cdot (NDVI_{loss,i}) \cdot P_v \cdot R_v
$$

where:
- $A_i$ - cell area i (ha)
- $NDVI_{loss,i}$ - loss of NDVI index in cell i (0-1)
- $P_v$ - base vegetation restoration cost ($/ha)
- $R_v$ - regional coefficient (1.0-2.5)

#### 2. Soil Restoration Cost (C‚Çõ)

$$
C_s = \sum_{i=1}^{n} A_i \cdot (1 - SSI_i) \cdot P_s \cdot D_i
$$

where:
- $SSI_i$ - soil stability index (0-1)
- $P_s$ - base soil reclamation cost ($/ha)
- $D_i$ - damage depth coefficient (1.0-3.0)

#### 3. Pollution Cleanup Cost (C‚Çö)

$$
C_p = \sum_{i=1}^{n} A_i \cdot C_{fuel} \cdot T_i \cdot K_{tox}
$$

where:
- $C_{fuel}$ - concentration of fuel residues (kg/ha)
- $T_i$ - component toxicity (1-10)
- $K_{tox}$ - cleanup cost coefficient ($/kg)

#### 4. Total Restoration Cost

$$
C_{total} = C_v + C_s + C_p + C_m + C_{indirect}
$$

where $C_m$ is the mechanical restoration cost, $C_{indirect}$ are indirect losses.

Restoration cost is adjusted considering:
- **Inflation**: Consumer Price Index (CPI) is used
- **Technological Progress**: Cost reduction of some methods
- **Resource Scarcity**: Cost increase in case of deficit

$$
P_t = P_0 \cdot (1 + r)^t \cdot TCF
$$

where:
- $P_t$ - cost in year t
- $P_0$ - base cost
- $r$ - annual inflation
- $t$ - number of years
- $TCF$ - technological correction factor

## üè¶ Application in Insurance

### Pricing Model

Insurance premium is calculated as:

$$
Premium = EL + Risk\ Loading + Expenses + Profit
$$

where:
- $EL$ (Expected Loss) = $\sum P_i \times L_i$
- $P_i$ - probability of i-th scenario (from Monte Carlo simulation)
- $L_i$ - loss amount in i-th scenario

### Premium Calculation Example

```python
def calculate_premium(launch_params, coverage_amount):
    """Calculates insurance premium for a launch."""
    
    # Simulate scenarios
    scenarios = monte_carlo_simulation(launch_params, n=1000)
    
    # Calculate expected losses
    expected_losses = []
    for scenario in scenarios:
        loss = calculate_ecological_damage(scenario)
        expected_losses.append(loss)
    
    # Statistical indicators
    el = np.mean(expected_losses)  # Expected loss
    std = np.std(expected_losses)  # Standard deviation
    
    # Premium calculation
    risk_loading = 1.5 * std  # Risk loading (1.5œÉ)
    expenses = 0.15 * coverage_amount  # Administrative expenses
    profit_margin = 0.10 * (el + risk_loading + expenses)  # Profit
    
    premium = el + risk_loading + expenses + profit_margin
    
    return {
        'expected_loss': el,
        'risk_loading': risk_loading,
        'premium': premium,
        'loss_distribution': expected_losses
    }
```
